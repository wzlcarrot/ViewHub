# 智能聊天机器人测试指南

## 快速测试步骤

### 1. 启动项目
```bash
cd b_version1/easylive-front/easylive-front-web
npm run dev
```

### 2. 访问搜索页面
- 在浏览器中打开项目
- 在搜索框输入任意关键词（如"编程"、"教程"等）
- 点击搜索按钮

### 3. 查看聊天机器人
- 搜索后，右下角会出现智能聊天机器人窗口
- 机器人会自动发送消息，解释你的搜索内容

### 4. 测试交互功能

#### 测试1: 推荐视频
在聊天框输入：
- "推荐视频"
- "推荐编程视频"
- "推荐教程视频"

**预期结果：** 机器人会显示3个相关视频卡片

#### 测试2: 点击视频卡片
点击机器人推荐的任意视频卡片

**预期结果：** 跳转到视频详情页

#### 测试3: 基本对话
在聊天框输入：
- "你好"
- "帮助"
- "怎么用"

**预期结果：** 机器人会给出相应的回复

#### 测试4: 最小化/展开
点击聊天窗口的标题栏

**预期结果：** 窗口会最小化或展开

## 常见问题排查

### 问题1: 聊天机器人不显示
**原因：** 组件导入失败
**解决：** 检查 `ChatBot.vue` 文件是否存在于 `src/components/` 目录

### 问题2: 推荐视频失败
**原因：** 后端API未启动
**解决：** 
- 检查后端服务是否运行
- 查看浏览器控制台是否有API错误
- 机器人会显示错误提示消息

### 问题3: 样式显示异常
**原因：** CSS未正确加载
**解决：** 
- 清除浏览器缓存
- 重启开发服务器
- 检查 `<style lang="scss" scoped>` 标签
